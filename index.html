About Me

Hi, I'm Abomi Daba, a software developer based in the Seattle area. My first interaction with code was in an intro to computer programming course at South Seattle College. I learned an overview of computer program design and problem solving with a focus on problem analysis, program development, testing and debugging. After I realized code can offer simple program solutions to real world problems, I decided to enroll in a program at Coding Dojo to increase my knowledge in software development! I completed my program and am set to graduate from South Seattle College with an associate in computer science in June 2023. I will be pursuing my Bachelors degree in Software Development in fall of 2023. When I am not working on an application or learning new technologies, you can catch me shooting hoops with some friends or my wife (we love basketball!), playing at the park with the kids, or playing board games with family!
